## About Me
I am a software developer specialized in Discord bots and AI integration. My main project is **Spritle**, a Discord bot designed for server management, automation, and AI-powered features.  
Currently working on **The Spritle Upgrade** with smarter automations and enhanced AI moderation tools.

## Spritle Overview
- Advanced **anti-raid & moderation systems**
- **Ticket and support management**
- **Multilanguage support** (English & Spanish)
- **Custom commands & automation**
- **AI-powered features**
